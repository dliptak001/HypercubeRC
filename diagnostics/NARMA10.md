# NARMA-10 Results

## What is NARMA-10?

NARMA-10 (Nonlinear AutoRegressive Moving Average, order 10) is the standard reservoir
computing benchmark for combined memory and nonlinear computation. The target sequence
is generated by the recurrence:

    y(t+1) = 0.3*y(t) + 0.05*y(t)*sum(y(t-i), i=0..9) + 1.5*u(t-9)*u(t) + 0.1

where u(t) is a random input in [0, 0.5]. This task is harder than Mackey-Glass because
it demands three capabilities simultaneously: memory of at least 10 steps (the u(t-9)
term), nonlinear computation (product terms y*sum and u*u), and temporal integration
(the autoregressive y(t) feedback). A reservoir that can only remember OR only compute
nonlinearly will fail — it must do both.

Standard ESN NRMSE on NARMA-10 ranges from 0.2 to 0.4. Lower is better.

## Findings

- **Translation layer dominates at DIM >= 7.** Improvement scales dramatically: -12%
  at DIM=7, -34% at DIM=8, -55% at DIM=9, -64% at DIM=10. The quadratic features
  (x², x*x') are essential for the nonlinear product terms in the NARMA-10 recurrence.
- **DIM 5-6: translation hurts or is neutral.** At DIM=5, NRMSE actually worsens by 9%
  with translation — the reservoir is too small for 80 features (overfitting). DIM=6 is
  flat. This matches the MackeyGlass pattern: translation only helps when the reservoir
  is large enough to support the expanded feature set.
- **DIM=10 with translation (0.148) beats the standard ESN range** (0.2-0.4) decisively.
  This is a strong result for a sparse reservoir with only 20 connections per vertex.
- **Raw features plateau at DIM 8-10** (0.40-0.45). Without translation, larger reservoirs
  don't improve much on NARMA-10 because the linear readout can't access the nonlinear
  state interactions needed for the product terms.
